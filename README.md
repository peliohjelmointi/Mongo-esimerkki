# Mongo-esimerkki

Kloonaa /Forkkaa projekti itsellesi ja asenna npm install -komennolla tarvittavat moduulit.
Projekti on tunneilla tehty ja ei ole mikään täydellinen tai tuotantokelpoinen, ja voi sisältää virheitä ja huonoja ratkaisuita.

Voit kirjautua /login -reitissä (johon / -reitti ohjaa) seuraavasti:
käyttäjänimi:joulupukki
salasana:joulu

Käyttäjänimellä on vain luku-oikeudet, eli et voi tallentaa tai päivittää blogeja, mutta näet, miten ohjelma muuten toimii koodeineen.
